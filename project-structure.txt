Tecnologias Principais
----------------------
Frontend:
- React 18 + TypeScript
- Vite (build tool)
- Wouter (routing)
- TanStack Query v5 (state management)
- React Hook Form + Zod (forms & validation)
- Tailwind CSS + Shadcn UI (styling)
- Axios (HTTP client)

Backend:
- Express.js
- Zod (validation)
- In-memory storage (pronto para PostgreSQL)

Tipografia:
- Inter (fonte principal)
- Space Grotesk (acentos/títulos)
