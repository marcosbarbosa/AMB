================================================================================
                         ROTAS DA APLICAÇÃO
================================================================================

PÁGINAS PÚBLICAS:
  /                          → Home (página inicial)
  /contato                   → Formulário de contato
  /login                     → Login de usuários
  /cadastro                  → Cadastro de novos usuários
  /esqueci-senha             → Recuperação de senha
  /redefinir-senha           → Redefinir senha
  /parceiros                 → Página de parceiros

ÁREA DO USUÁRIO (requer autenticação):
  /painel                    → Painel do usuário
  /editar-perfil             → Editar perfil

ÁREA ADMINISTRATIVA (requer autenticação admin):
  /admin                     → Painel administrativo
  /admin/eventos             → Gestão de eventos (CRUD)
  /admin/eventos/inscricoes/:id → Inscrições de times no evento
  /admin/eventos/conteudo/:id   → Posts e boletins do evento
  /admin/times               → Gestão de times
  /admin/jogos               → Gestão de jogos
  /admin/associados          → Gestão de associados
  /admin/parceiros           → Gestão de parceiros
  /admin/placar              → Gestão de placares

================================================================================
                    ENDPOINTS DA API (./server/routes.ts)
================================================================================

POST   /api/contact           → Enviar mensagem de contato
GET    /api/contact           → Listar mensagens de contato
GET    /api/contact/:id       → Obter mensagem por ID

(Demais endpoints são gerenciados pelo backend PHP externo no HostGator)

================================================================================
                       TECNOLOGIAS DO PROJETO
================================================================================

FRONTEND:
  • React 18 + TypeScript
  • Vite 5.x (build tool)
  • React Router DOM 6.x (roteamento)
  • TanStack Query v5 (gerenciamento de estado/cache)
  • Tailwind CSS 3.x + Shadcn UI (estilização)
  • React Hook Form + Zod (validação de formulários)
  • Axios (requisições HTTP)
  • Lucide React (ícones)
  • Framer Motion (animações)
  • Swiper/Embla (carrosséis)

BACKEND:
  • Express.js 4.x
  • TypeScript
  • Drizzle ORM
  • PostgreSQL (Neon - banco de dados)
  • Zod (validação de dados)

INFRAESTRUTURA:
  • Replit (hospedagem desenvolvimento)
  • HostGator (backend PHP externo - produção)
