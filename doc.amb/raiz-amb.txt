================================================================================
                           ROTAS DA APLICAÇÃO
================================================================================

PÁGINAS PÚBLICAS:
  /                     → Home (página inicial)
  /contato              → Formulário de contato
  /login                → Login de usuários
  /cadastro             → Cadastro de novos usuários

ÁREA ADMINISTRATIVA (requer autenticação):
  /admin                → Painel administrativo
  /admin/eventos        → Gestão de eventos (CRUD)
  /admin/eventos/inscricoes/:id → Inscrições de times no evento
  /admin/eventos/conteudo/:id   → Posts e boletins do evento
  /admin/times          → Gestão de times
  /admin/jogos          → Gestão de jogos
  /admin/associados     → Gestão de associados

================================================================================
                           TECNOLOGIAS UTILIZADAS
================================================================================

FRONTEND:
  - React 18 + TypeScript
  - Vite (build tool)
  - React Router DOM (roteamento)
  - TanStack Query v5 (gerenciamento de estado)
  - Tailwind CSS + Shadcn UI (estilização)
  - React Hook Form + Zod (formulários)
  - Axios (requisições HTTP)
  - Lucide React (ícones)
  - Framer Motion (animações)

BACKEND:
  - Express.js
  - TypeScript
  - Drizzle ORM
  - PostgreSQL (Neon)
  - Zod (validação)

INFRAESTRUTURA:
  - Replit (hospedagem dev)
  - HostGator (backend PHP externo)
